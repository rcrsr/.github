Open Source projects by [Andre Bremer](https://github.com/AndreBremer).

## Claude Code Plugins

- [claude-plugins](https://github.com/rcrsr/claude-plugins) — Plugin marketplace registry for Claude Code
  - [checkmate](https://github.com/rcrsr/checkmate) — Automated linting, formatting, and type checks on file edits
  - [snoop](https://github.com/rcrsr/snoop) — Transcript capture with built-in anti-pattern detection
  - [policies](https://github.com/rcrsr/claude-plugins/tree/main/policies) — Auto-injects policy sections into subagents via § notation

## Infrastructure

- [claude-code-runner](https://github.com/rcrsr/claude-code-runner) — Deterministic, scripted, unattended Claude Code execution
- [mcp-policy-server](https://github.com/rcrsr/mcp-policy-server) — MCP server for § notation policy fetching with O(1) section lookups
